About Me

꼼꼼함과 추진력을 바탕으로
웹 개발에 몰입하는 개발자입니다.

사용자 경험을 최우선으로 생각하며, 깨끗하고 유지보수가 쉬운 코드를 작성합니다. 끊임없이 새로운 기술을 학습하고, 실무 프로젝트에 적용하는 것을 즐깁니다.

현재 전북우리사이 웹사이트를 개발·관리하며 실무 경험을 쌓고 있습니다. HTML, CSS, JavaScript를 기반으로 반응형 웹사이트를 만들며, PHP와 MySQL을 활용한 백엔드 연동 경험도 보유하고 있습니다.

Frontend

  • HTML5 웹 표준을 준수하며 시맨틱 태그를 활용한 구조화된 마크업을 작성할 수 있습니다.
  • CSS3 Flexbox, Grid를 활용한 반응형 레이아웃 설계 및 애니메이션 구현이 가능합니다.
  • JavaScript function 기능을 이용하여 슬라이드 동작과 토글, 팝업 등 간단한 이벤트를 사용할 수 있습니다.
  • Responsive Design 모든 디바이스에서 최적화된 사용자 경험을 제공하는 반응형 웹을 제작합니다.

Backend

  • PHP 서버 사이드 로직 구현 및 데이터베이스 연동을 통한 동적 웹사이트 개발이 가능합니다.
  • MySQL 데이터베이스 설계, 쿼리 최적화 및 효율적인 데이터 관리를 수행합니다.
  • CMS (Gnuboard) 그누보드를 활용한 커뮤니티 사이트 구축 및 커스터마이징 경험이 있습니다.

Tools

  • Git / GitHub 버전 관리를 활용한 협업 및 코드 이력 관리를 능숙하게 수행합니다.
  • VS Code 확장 프로그램을 활용한 효율적인 개발 환경을 구축하여 사용합니다.
  • Figma 디자인 시안을 분석하고 정확하게 웹으로 구현하는 작업을 수행합니다.
  • Photoshop 이미지 편집, 디자인 작업 및 웹 최적화 이미지를 제작합니다.
  • Illustrator 벡터 그래픽 제작 및 로고, 아이콘 디자인 작업을 수행합니다.
  • InDesign 편집 디자인 및 인쇄물 레이아웃 작업을 제작합니다.
  • 한글 / Excel / PowerPoint 문서 작성, 데이터 분석 및 프레젠테이션 자료 제작이 가능합니다.
  • Chrome DevTools 디버깅, 성능 최적화 및 반응형 테스트를 효과적으로 진행합니다.
